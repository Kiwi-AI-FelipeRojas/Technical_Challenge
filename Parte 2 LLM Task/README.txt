Part 2 LLM Task - Structured Data Extraction & Processing

This folder contains the solution for the second task of the challenge. The task was solved using Azure OpenAI to analyze all the documents stored in the "Input_files" folder. The files can be either Word or PDF files. In order to run the code it is necessary fill all the Azure credentials in the "Azure_credencials.txt" file. The file "Extractor.ipynb" contains the code used to solve the task. When the code is runed, the file called "extractor_results.txt" will be created. In it a JSON is stored containing the extracted data of each of the documents.
